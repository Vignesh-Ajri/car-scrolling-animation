# Scroll Car Animation

A scroll-based car animation built using **GSAP** and **ScrollTrigger**.  
The car moves across the screen as you scroll, revealing animated text and a growing trail.

Live Demo:  
[https://vignesh-ajri.github.io/car-scrolling-animation/](https://vignesh-ajri.github.io/car-scrolling-animation/)

## Tech Used

HTML • CSS • JavaScript • GSAP
